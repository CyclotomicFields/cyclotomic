pub struct Number {
    order: u64,
    coeffs: HashMap<u64, BigRational>,
}

Represents a polynomial in the orderth root of unity.

Simplest possible choice of data structure - a hash map. We assume only that each term has an exponent less than the order. So the only real term is the one for exponent zero, for example.
